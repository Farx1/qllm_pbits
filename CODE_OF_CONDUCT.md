# Code of Conduct

## Summary
We want this repository to be a respectful place to learn, discuss, and collaborate.

## Be respectful
- Assume good intent.
- Be kind and constructive.
- Critique ideas, not people.

## Unacceptable behavior
- Harassment, insults, or discriminatory comments.
- Trolling, personal attacks, or sustained disruption.
- Sharing others' private information without permission.

## Reporting
If something feels off, open an issue or contact the maintainer privately.

## Enforcement
The maintainer may remove, edit, or reject contributions that are not aligned with this Code of Conduct.
